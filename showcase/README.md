# Showcase

This directory contains the static project homepage for Agent Restaurant.

It is here to explain the open-source skill, the 1.0 reward rules, and the food menu in the project's pixel-terminal visual language.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## What It Demonstrates

- the project positioning for Agent Restaurant
- the version 1.0 token-to-coin rules
- the product traits of the reusable skill
- the pixel-art reward menu with coin prices
